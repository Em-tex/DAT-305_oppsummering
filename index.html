Module 1: Introduction to Machine Learning

What is Machine Learning?

Unlike traditional programming (explicit rules), Machine Learning (ML) involves learning patterns from data.

Types of Learning

Supervised Learning

Learning with labeled data.

  • Regression: Predicting continuous values (e.g., house prices, temperature).
  • Classification: Predicting categories (e.g., spam vs. not spam).

Unsupervised Learning

Learning with unlabeled data. Finding hidden structures.

  • Clustering: Grouping similar data (e.g., customer segmentation).
  • Dimensionality Reduction: Simplifying data (e.g., PCA).

Python Libraries Stack

  • NumPy: N-dimensional arrays, efficient math computation.
  • Pandas: Data structures (DataFrames), data manipulation and analysis.
  • Scikit-Learn: Algorithms for ML (Classification, Regression, Clustering). Tools like `train_test_split`.

Module 2: Mathematics for AI

Linear Algebra

  • Vector: 1D array of numbers. Has both magnitude (length) and direction. Used to represent features.
  • Matrix: 2D array. Rows = samples, Columns = features.
  • Dot Product: A measure of similarity between two vectors. If the dot product is 0, the vectors are orthogonal.

Data Representation

  • One-Hot Encoding: Converting categories into binary vectors. Results in sparse vectors.
  • Embeddings: Dense vectors where words with similar meanings are located close to each other in the vector space.

Module 3: Machine Learning Algorithms

Evaluation Metrics

How do we know if the model works?

For Regression:
  • MSE (Mean Squared Error): The average of the squared differences between predicted and actual values. Punishes large errors.
  • R-Squared: How well the data fits the regression line (0 to 1).
For Classification:
  • Accuracy: Correct Predictions / Total Predictions.
  • Precision: Out of those we predicted as positive, how many were actually positive?
  • Recall: Out of those that were actually positive, how many did we find?
  • F1-Score: The harmonic mean of Precision and Recall.

Module 4: Artificial Neural Networks (ANN)

The Neuron (Perceptron)

Inspired by biological neurons. Inputs ($x$) are multiplied by weights ($w$), summed with a bias ($b$), and passed through an activation function.

$$z = \sum (x_i \cdot w_i) + b \rightarrow \sigma(z)$$

Backpropagation

The algorithm used to train the network:

  1. Forward Pass: Calculate predictions.
  2. Calculate Loss: Determine the error.
  3. Backward Pass: Propagate the error back to find gradients (which weights contributed to the error).
  4. Update Weights: Use Gradient Descent to adjust weights.

Module 5: Deep Learning

CNN (Convolutional Neural Networks)

Used for Images. Key components:

  • Convolution Layer: Uses filters/kernels to extract features (edges, shapes).
  • Pooling Layer: Reduces spatial dimensions (downsampling) to reduce computation.

RNN & LSTM (Sequential Data)

Standard RNNs suffer from the Vanishing Gradient problem.

LSTM (Long Short-Term Memory) solves this by using "gates" (Input, Output, Forget) to regulate what information to keep or discard.

Module 6: NLP

Preprocessing & Features

  • Tokenization: Splitting text into words or sub-words.
  • TF-IDF: Measures word importance (High if frequent in doc, low if frequent in all docs).

Transformers

State-of-the-art for NLP (e.g., BERT, GPT).

Key innovation is Self-Attention.

Hugging Face: A library and hub for pre-trained transformer models.
