Module 2: Mathematics for AI

Linear Algebra, Data Representation & PCA

2.1 Vectors & Operations

Vector: Magnitude & Direction. Represents a data point's features.

Dot Product: $\mathbf{a} \cdot \mathbf{b} = \sum a_i b_i$.

Crucial: If Dot Product is 0, vectors are Orthogonal (Uncorrelated).

[ 2, 5 ] · [ 3, 1 ] = (2*3) + (5*1) = 11

2.2 Dimensionality Reduction (PCA)

Principal Component Analysis allows us to visualize high-dimensional data and remove noise.

1. Standardize
Scale data (Mean=0, Var=1) so large units don't dominate.
2. Covariance Matrix
Calculate relationships between variables.
3. Eigen-Decomposition
Find Eigenvectors (Direction) & Eigenvalues (Magnitude/Variance).
4. Projection
Project data onto the top $K$ eigenvectors.

Why do we need SVD?

Singular Value Decomposition (SVD) is a matrix factorization method ($M = U \Sigma V^T$). It is numerically more stable and efficient for calculating PCA than calculating the covariance matrix directly.

Module 3: Algorithms

Classic Machine Learning Models & Evaluation

3.1 The Algorithms

Linear Regression

Fitting a line. Uses MSE (Mean Squared Error) as the cost function.

$$ J = \frac{1}{n} \sum (y_{pred} - y_{actual})^2 $$

KNN (K-Nearest Neighbors)

Lazy Learner. No training. Memorizes data. Predicts based on neighbors.

K-Factor: Small K = Overfitting (Noise). Large K = Underfitting.

SVM (Support Vector Machine)

Finds the Hyperplane with the max Margin.

Kernel Trick: Maps non-linear data to higher dimensions to make it separable.

Decision Trees

Splits data to reduce Entropy (Randomness).

Goal: High Information Gain.

Deep Dive: Accuracy vs F1-Score

Accuracy ($(TP+TN)/Total$) is misleading if you have 99% healthy people and 1% sick. Predicting "Healthy" for everyone gives 99% accuracy but kills the patient.

Recall ($TP / (TP+FN)$) is vital for medical/safety.

Precision ($TP / (TP+FP)$) is vital for spam filters.

F1-Score is the harmonic mean of both.

3.3 Clustering (K-Means)

Iterative algorithm:

  1. Init K Centroids.
  2. Assign points to nearest centroid.
  3. Update centroid to mean of points.
  4. Repeat.

Elbow Method: Plot Inertia vs K to find the optimal number of clusters.

4.2 Activation Functions

The secret sauce of Deep Learning. They introduce non-linearity.

FuncRangeDescription
Sigmoid(0, 1)Classic. Good for probabilities. Bad: Vanishing Gradient in deep nets.
Tanh(-1, 1)Zero-centered. Still suffers Vanishing Gradient.
ReLU[0, inf)Standard. $max(0, x)$. Fast, solves vanishing gradient.
Softmax(0, 1)Used in Output Layer for multi-class. Sums to 1.

4.3 Training: Gradient Descent

Imagine standing on a misty mountain (High Loss). You want to reach the valley (Low Loss).

1. Forward Pass
Calculate prediction.
2. Loss Calc
Compare to ground truth.
3. Backpropagation
Calculate gradients (direction).
4. Update
Weights = Weights - (Learning Rate * Gradient).

What is Vanishing Gradient?

When using Sigmoid, derivatives are small (< 0.25). As you multiply them backwards through layers (Chain Rule), the gradient becomes tiny. The early layers stop learning. Solution: Use ReLU.

5.2 RNN & LSTM

Specialized for Sequential Data (Text, Time).

Problem: RNNs have "Short-term memory" due to Vanishing Gradient.

Solution: LSTM (Long Short-Term Memory).

1. Forget Gate

Sigmoid. Decides what to throw away.

2. Input Gate

Sigmoid+Tanh. Decides what to store.

3. Output Gate

Sigmoid. Decides what to pass on.

5.3 Autoencoders

Unsupervised learning for compression/reconstruction.

Input → Encoder → Bottleneck (Z) → Decoder → Output

The Bottleneck forces the network to learn the most efficient representation (Latent Space).

Module 6: NLP

Natural Language Processing & Transformers

6.1 Text Preprocessing Pipeline

1. Tokenization
Splitting text into words/units.
2. Cleaning
Lowercasing, removing Stop Words (the, is, a).
3. Normalization
Stemming (Chop: Fishing->Fish) or Lemmatization (Root: Better->Good).
4. Vectorization
Converting text to numbers (TF-IDF, Word2Vec).

6.2 Vectorization Techniques

TF-IDF

Term Frequency - Inverse Document Frequency.

Highlights words that are frequent in this document but rare in the corpus (Keywords).

Word Embeddings (Word2Vec)

Dense vectors based on context.

Allows semantic math: $King - Man + Woman = Queen$.

6.3 Transformers (BERT & GPT)

Based on Self-Attention: The model looks at the whole sentence at once.

BERT

Encoder-only. Bidirectional. Good for understanding context (Classification, Q&A).

GPT

Decoder-only. Autoregressive. Good for generating text (predicting next word).
