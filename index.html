Module 1: Introduction to Machine Learning

Foundations, Types of Learning, and Python Ecosystem

1.1 What is Machine Learning?

Traditional programming uses explicit rules. Machine Learning (ML) uses data to learn those rules.

Supervised

Learning with Labeled Data.

  • Regression
  • Classification
Unsupervised

Learning with Unlabeled Data.

  • Clustering
  • Dim. Reduction
Reinforcement

Agent learns via Rewards.

Module 2: Mathematics for AI

Linear Algebra & Dimensionality Reduction

2.1 Vectors and Matrices

Dot Product: If 0, vectors are Orthogonal (Uncorrelated).

[ 1, 2 ] · [ 3, 4 ] = 11

2.2 Data Representation

One-Hot: Sparse, binary vectors.

Embeddings: Dense, semantic vectors.

2.3 PCA

Reduces dimensions by finding Eigenvectors (Direction) and Eigenvalues (Variance).

Module 3: ML Algorithms

Regression, Classification & Clustering

3.1 Algorithms

KNN (K-Nearest Neighbors)

Classifies based on $K$ neighbors. "Lazy Learner".

Decision Trees

Splits data to reduce Entropy (Impurity).

SVM

Maximizes Margin. Uses Kernels for non-linear data.

3.2 Metrics

Pred +Pred -
Actual +TPFN
Actual -FPTN

Accuracy: Overall correct. Recall: TP/(TP+FN). Precision: TP/(TP+FP).

3.3 K-Means

Unsupervised. Minimizes distance to centroids.

Module 4: Neural Networks

Perceptrons & Backpropagation

4.1 The Neuron

x
→ w →
$\sum$

$$Output = f(\sum (x \cdot w) + b)$$

4.2 Activation Functions

  • Sigmoid: (0,1). Problem: Vanishing Gradient.
  • ReLU: max(0, z). Solves Vanishing Gradient.
  • Softmax: Multi-class probabilities.

4.3 Training

Backpropagation: Calculates gradients.

Gradient Descent: Updates weights.

Epoch: Full dataset pass.

Module 5: Deep Learning

5.1 CNN (Images)

Convolution: Extracts features using Filters.

Pooling: Reduces size (Down-sampling).

5.2 RNN & LSTM

RNN: Sequence memory. Issue: Vanishing Gradient.

LSTM: Solves it using Gates (Input, Output, Forget).

5.3 Autoencoders

Input → Encoder → Bottleneck → Decoder → Output

6.1 Basics
