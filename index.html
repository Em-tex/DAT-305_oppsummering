The Core Difference:

  • Traditional Programming: Rules + Data = Answers.

1.2 Types of Learning

Supervised

Learning with a "Teacher" (Labels).

  • Regression: Predicting numbers (Prices, Temp).
  • Classification: Predicting classes (Spam/Ham).

Unsupervised

Self-discovery (No Labels).

  • Clustering: Grouping similar items (K-Means).
  • Dim. Reduction: Compressing data (PCA).

Reinforcement

Learning by Interaction.

  • Agent: The learner.
  • Reward: Feedback signal.
  • Goal: Maximize cumulative reward.

Module 2: Mathematics for AI

Linear Algebra, Data Representation & PCA

2.1 Vectors & Operations

Vector: Magnitude & Direction. Represents a data point's features.

Dot Product: $\mathbf{a} \cdot \mathbf{b} = \sum a_i b_i$.

Crucial: If Dot Product is 0, vectors are Orthogonal (Uncorrelated).

[ 2, 5 ] · [ 3, 1 ] = (2*3) + (5*1) = 11

2.2 Dimensionality Reduction (PCA)

Principal Component Analysis allows us to visualize high-dimensional data and remove noise.

1. Standardize
Scale data (Mean=0, Var=1) so large units don't dominate.
2. Covariance Matrix
Calculate relationships between variables.
3. Eigen-Decomposition
Find Eigenvectors (Direction) & Eigenvalues (Magnitude/Variance).
4. Projection
Project data onto the top $K$ eigenvectors.

Module 3: Algorithms

Classic Machine Learning Models & Evaluation

3.1 The Algorithms

Linear Regression

Fitting a line. Uses MSE (Mean Squared Error).

KNN (K-Nearest Neighbors)

Lazy Learner. Predicts based on neighbors. Small K = Overfitting.

SVM (Support Vector Machine)

Finds Hyperplane with max Margin. Uses Kernel Trick for non-linear data.

Decision Trees

Splits data to reduce Entropy. Goal: High Information Gain.

4.2 Activation Functions

  • Sigmoid: (0 to 1). Problem: Vanishing Gradient.
  • ReLU: (0 to inf). Solves Vanishing Gradient. Standard.
  • Softmax: Output layer for multi-class prob.

Module 5: Deep Learning

CNN, RNN, LSTM & Autoencoders

5.1 CNN (Images)

Uses Convolutions (Filters) to extract features and Pooling to reduce size.

5.2 RNN & LSTM (Text/Time)

RNN: Has memory but suffers Vanishing Gradient (forgets long term).

LSTM: Uses Gates (Forget, Input, Output) to fix memory.

7.3 Keras (Deep Learning)

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# 1. Initialize
model = Sequential()

# 2. Add Layers (Input + Hidden)
# Dense = Fully Connected. 
# 128 neurons, ReLU activation.
model.add(Dense(128, activation='relu', input_shape=(784,)))

# 3. Output Layer (10 classes)
model.add(Dense(10, activation='softmax'))

# 4. Compile (Loss & Optimizer)
model.compile(optimizer='adam', loss='categorical_crossentropy')

# 5. Train
model.fit(X_train, y_train, epochs=5)
