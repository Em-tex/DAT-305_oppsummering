Module 2: Mathematics for AI

Linear Algebra & PCA (Principal Component Analysis)

2.1 Vectors: The Language of AI

Dot Product

$\mathbf{a} \cdot \mathbf{b} = \sum a_i b_i$

This number tells us how similar two vectors are.

  • Positive: Pointing same direction (Similar).
  • 0: Orthogonal (Perpendicular/Independent). No correlation.
  • Negative: Pointing opposite direction.
Orthogonal (0 similarity)

2.2 PCA: Principal Component Analysis

Why? To simplify data (remove noise) without losing key info. Compressing many columns into a few "Principal Components".

1. Standardize
Scale data (Mean=0, Variance=1) so large numbers (e.g. Salary) don't dominate small ones (e.g. Age).
2. Covariance Matrix
Calculate how variables vary together (Correlation).
3. Eigen-Decomposition
Find Eigenvectors (Directions with most info) and Eigenvalues (Importance).
4. Project
Project data points onto the new axes (Components).

SVD (Singular Value Decomposition) is a matrix factorization method often used to calculate this efficiently.

Module 4: ANN (Artificial Neural Networks)

Neural Networks, Backprop & Gradient Descent

4.1 How does it learn? (Gradient Descent)

The process of minimizing the error (Loss) is called Gradient Descent.

  • Loss Function (J): Tells us how wrong the model is (e.g., MSE).
  • Gradient: The slope. Tells us which direction to adjust weights to reduce error.
  • Learning Rate ($\alpha$): Step size down the hill.

Goal: Roll the ball to the bottom (Global Minima)

4.2 Activation Functions

They introduce non-linearity. Without them, a Neural Net is just Linear Regression.

  • Sigmoid: S-shape (0 to 1). Old school. Problem: Vanishing Gradient.
  • ReLU (Rectified Linear Unit): $max(0, x)$. Standard for hidden layers. Solves Vanishing Gradient.
  • Softmax: Used in Output Layer for multi-class probabilities (sums to 100%).

Module 5: Deep Learning (DL)

CNN, RNN, LSTM & Autoencoders

5.1 CNN (Convolutional Neural Networks)

Specialized for images. Instead of pixels, it looks for patterns (edges, shapes).

Convolution (Filter)

A small window slides over the image to extract features. Preserves spatial structure.

Pooling (Subsampling)

Reduces image size by keeping only important features (e.g. Max Pooling). Reduces computation.

5.2 RNN & LSTM (Sequences)

For text, audio, and time-series. Data where order matters.

RNN (Recurrent Neural Network)

Has a "loop" to remember the previous step. Suffers from short-term memory.

LSTM (Long Short-Term Memory)

Advanced RNN. Uses Gates to decide what to keep and what to forget. Solves Vanishing Gradient.

Module 6: NLP (Natural Language Processing)

Text Analysis & Transformers

6.1 From Words to Numbers

1. Tokenization
Splitting text into units (words).
2. Stop Words
Removing noise words ("and", "the", "is").
3. Lemmatization
Finding the root form ("Running" → "Run"). Better than Stemming.
4. Vectorization
Converting words to vectors (TF-IDF or Embeddings).

6.2 The Transformer Revolution (BERT & GPT)

Key Tech: Self-Attention. The model reads the whole sentence at once to understand context.

BERT

Bidirectional Encoder Representations from Transformers

Reads text both ways. Great for understanding context and classification.

GPT

Generative Pre-trained Transformer

Reads forward only. Predicts the next word. Great for generating text.
