# DAT305 - AI for Engineers: Ultimate Exam Prep

Dette er en komplett repetisjonsportal for emnet DAT305. Den dekker alle 6 moduler fra pensum, inkludert matematikk, ML-algoritmer, Deep Learning og NLP.

## Oppsett
1.  Last ned filene `index.html`, `style.css`, `script.js`, og `questions.js` til samme mappe.
2.  Åpne `index.html` i nettleseren din.

## Funksjoner
* **Full pensumdekning:** Tekstlige sammendrag av alle slides.
* **Visuelle hjelpemidler:** Plassholdere for diagrammer og CSS-baserte flytskjema.
* **Eksamenssimulator:** En quiz med 60+ spørsmål som trekkes tilfeldig hver gang.
* **Umiddelbar feedback:** Forklaringer på hvert spørsmål for dypere læring.

## Moduler
1.  **Intro:** AI vs ML, Supervised/Unsupervised, Python libraries.
2.  **Math:** Lineær algebra, Vektorer, PCA.
3.  **ML Algorithms:** Regresjon, KNN, SVM, Evaluering.
4.  **ANN:** Nevroner, Backpropagation, Gradient Descent.
5.  **Deep Learning:** CNN, RNN, LSTM, Autoencoders.
6.  **NLP:** Tokenization, TF-IDF, Transformers (BERT/GPT).
